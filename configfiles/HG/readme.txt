To Flash Huawei HW553/HW556a
----------------------------

Hold in Reset button while powering on the router.

Hold for 30 seconds then release.

Use your web browser to go to 192.168.1.1

Upload the ROOter image file and flash the router.

Wait for the router to reboot (Power Led is red).


Identifying HW556a Version
-------------------------

Serial number (first 5 digits) for

Version A - 30462, 30605, 30608

Version B - 30562, 30692, 30693, 31300, 30634 

Version C - 30695, 30694, 31301, 31507, 30555, 31525, 31935, 31901, 31902

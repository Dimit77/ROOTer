Gl.iNet Flashing
----------------

Method 1
--------

Log on the factory Web UI and go to Advanced Settings at top right of screen.

This takes you to the OpenWRT web interface and you can flash as normal.

Method 2
--------

Log on the factory Web UI and change the IP Address of the LAN to 192.168.1.1

Reboot the router and it will come up with the OpenWRT web interface
at 192.168.1.1.

Flash as normal.
